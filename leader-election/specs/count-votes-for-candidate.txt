=== TASK_DESCRIPTION ===
Implement a pure function that counts how many votes a given candidate received in an election round.

Input: a list of Nat votes (each element is the node ID the voter voted for; the same ID may appear multiple times) and a Nat candidate ID.

Output: a Nat equal to the number of elements in the votes list that equal the candidate ID.

Preconditions: none beyond the types; the votes list may be empty and may contain duplicates.

Edge cases: empty votes list returns 0; candidate not present in the list returns 0; all votes for the candidate returns the list length.

Determinism: the function is pure; identical inputs always produce identical outputs.
=== METHOD_SIGNATURE ===
method countVotesForCandidate (votes : List Nat) (candidate : Nat) : Nat
