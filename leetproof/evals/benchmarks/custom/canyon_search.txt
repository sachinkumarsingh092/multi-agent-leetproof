=== TASK_DESCRIPTION ===
Write a method in Velvet called CanyonSearch that finds the minimum absolute difference between an element of two sorted nonempty integer arrays. The complexity should be O(n).

Given two arrays `a` and `b` of integers, each sorted in non-decreasing order and non-empty, consider all pairs of indices (i,j) with i in bounds of `a` and j in bounds of `b`. For each pair, compute the absolute difference |a[i] - b[j]| as a natural number using Int.natAbs. The output is the minimum such natural number among all pairs. The result must be achievable by some pair and be less than or equal to every pairwise distance.

Preconditions:
- Both arrays are non-empty (a.size > 0, b.size > 0)
- Both arrays are sorted in non-decreasing order

Example: a = [1,3,5], b = [2,4,6] => minimum distance is 1

=== METHOD_SIGNATURE ===
method CanyonSearch(a: Array Int, b: Array Int) returns (result: Nat)
