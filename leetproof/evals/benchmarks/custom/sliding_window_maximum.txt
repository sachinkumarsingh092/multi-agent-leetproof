=== TASK_DESCRIPTION ===
Write a method in Velvet called SlidingWindowMaximum that, given an array of integers and a window size k, returns an array containing the maximum value in each sliding window as it moves from left to right. The complexity should be O(n).

A sliding window of size k starts at position 0 and moves right one position at a time. The window stops when it reaches the end of the array (last window starts at index n-k). For each window position i (0 to n-k), find the maximum of nums[i..i+k-1]. The result array has length n-k+1 (number of valid window positions). Each element result[i] equals the maximum of the subarray nums[i..i+k-1].

Preconditions:
- k must be positive (k >= 1)
- k must not exceed array length (k <= nums.size)
- Array must be non-empty (nums.size >= 1)

Example: nums = [1,3,-1,-3,5,3,6,7], k = 3
- Window [1,3,-1] -> max = 3
- Window [3,-1,-3] -> max = 3
- Window [-1,-3,5] -> max = 5
- Window [-3,5,3] -> max = 5
- Window [5,3,6] -> max = 6
- Window [3,6,7] -> max = 7
- Result: [3,3,5,5,6,7]

=== METHOD_SIGNATURE ===
method SlidingWindowMaximum(nums: Array Int, k: Nat) returns (result: Array Int)
