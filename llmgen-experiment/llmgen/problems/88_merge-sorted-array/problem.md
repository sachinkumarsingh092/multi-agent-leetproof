# 88. Merge Sorted Arrays


You are given two integer arrays `nums1` and `nums2`, each sorted in **non-decreasing order**, with lengths `m` and `n` respectively.

Return a new array of length `m + n` that contains all elements from `nums1` and `nums2` merged in **non-decreasing order**.

**Example 1:**

```
Input:  nums1 = [1,2,3], nums2 = [2,5,6]
Output: [1,2,2,3,5,6]
```

**Example 2:**

```
Input:  nums1 = [1], nums2 = []
Output: [1]
```

**Example 3:**

```
Input:  nums1 = [], nums2 = [1]
Output: [1]
```

**Constraints:**

Your algorithm should run in **O(m + n)** time and **O(m + n)** extra space.
