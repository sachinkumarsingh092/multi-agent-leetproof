# 50. Pow(x, n, p)

Given three integers `x`, `n`, and `p`, compute `x` raised to the power `n` modulo `p` (i.e., `x^n mod p`).

**Example 1:**

```
Input: x = 2, n = 10, p = 1000
Output: 24
Explanation: 2^10 = 1024, and 1024 mod 1000 = 24.
```

**Example 2:**

```
Input: x = 3, n = 5, p = 13
Output: 9
Explanation: 3^5 = 243, and 243 mod 13 = 9.
```

**Example 3:**

```
Input: x = 7, n = 1, p = 7
Output: 0
```

**Constraints:**

Your algorithm should run in **O(log n) time and O(1) space**.
